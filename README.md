# 字符串分类管理器

这是一个使用 Dash 和 Dash Bootstrap Components 构建的字符串分类和选择应用。

## 功能

- 输入字符串并对其进行分类
- 保存分类后的字符串
- 按分类查看已保存的字符串
- 选择特定的字符串
- 按分类显示选中的字符串

## 安装和运行

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行应用：
```bash
python app.py
```

3. 在浏览器中打开 `http://127.0.0.1:8050/` 访问应用。

## 使用方法

1. **添加字符串**：
   - 在文本框中输入要分类的字符串
   - 在分类框中输入分类名称
   - 点击"添加字符串"按钮

2. **查看已保存的字符串**：
   - 使用下拉菜单选择特定分类或查看所有分类
   - 点击字符串卡片上的"选择"按钮来选择字符串

3. **查看选中的字符串**：
   - 选中的字符串会显示在右侧面板中
   - 字符串按分类组织显示
   - 点击"清除选择"按钮可以清除所有选中的字符串

## 数据存储

应用会将数据保存在 `string_data.json` 文件中，该文件位于应用根目录下。数据以 JSON 格式存储，包含所有分类和对应的字符串。
